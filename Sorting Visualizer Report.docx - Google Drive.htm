SORTING VISUALIZER REPORT 2

Overview

This is a sorting graphical user interface that is built with the Swing library.

What we have worked on as a group so far is created a Text box with the Swing UI tool

for entering the number of elements that are going to be sorted. We have a drop-down option

for which sorting methods that will be utilized for sorting the number of elements. Have a

separate amount of colors for the bar graph that is the number of elements. Mix drop-down

tool that mixes/randomize the elements and a drop-down that changes the colors of the

background image.
